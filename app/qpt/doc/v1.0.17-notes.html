QPT 1.0.17 Release Notes

QPT 1.0.17 addresses the issues described below. Also, please note the new download site indicated in the installation instructions. Release notes for the previous release are available here.

The QPT documentation index is available here.

To install and run this release of QPT:
First, uninstall the exisiting 2011A version → launcher -a:qpt-2011A -u
To install → launcher -i:qpt -a:qpt-2011A -h:build.cl.gemini.edu
To run → launcher -a:qpt-2011A
To uninstall → launcher -a:qpt-2011A -u
As always, the first time you run QPT it will download necessary components. This may take several minutes, especially in the north. Your previous installation of QPT should continue to function normally.
